# Strength Inventory
**Web application for comparing gym equipment, memberships, and opening hours.**

## Status
Early development with major parts missing.

## Stack
Frontend: **React** with **Vite**.

Backend: **Express** with **Sequelize** for ORM and **Zod** for runtime validation.

Compatible databases: PostgreSQL, SQLite, MySQL, MariaDB, and Oracle.

Node v24.13.0.
